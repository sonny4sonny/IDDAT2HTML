Conceptual Response

Prompt: Novel Input Technique: Location

This project is an audio-visualiser controlled by location and user input.

Latitude determines bright vs dark mood (colour and synth timbre), longitude sets tempo, and user drag across the screen bends pitch (X) and rhythmic density (Y).

This avoids traditional sliders or menus within the UI in pursuit of embodied, environmental interaction.

Technical Approach

Advanced method: navigator.geolocation.getCurrentPosition.

System map:
Tap → unlocks audio + location → Latitude → mode (bright/dark) → Longitude → tempo (BPM) → Drag X → pitch detune → Drag Y → note density → Tone.js (Synth + Kick) → Circle pulses with kick.
